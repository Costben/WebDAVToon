# WebDAVToon 功能白皮书 (Features Summary)

WebDAVToon 是一款专为 Android 平台设计的跨源漫画与图片阅读器。通过结合 Kotlin 的灵活性与 Rust 的高性能网络处理能力，它为用户提供了从本地到远程服务器的无缝阅读体验。

---

## 📂 1. 多源数据接入 (Multi-Source Access)

*   **本地存储 (Local Storage)**：支持浏览手机内部存储及 SD 卡中的图片文件夹。
*   **WebDAV 协议支持**：原生适配 AList、群晖 (Synology)、坚果云等主流 WebDAV 服务。
*   **SMB (Samba) 协议支持**：
    *   支持 SMB v2/v3 协议。
    *   **针对性修复**：在 Rust 层强制启用 NTLM 认证并禁用 Kerberos，完美解决 Android 系统上常见的 SMB 连接权限错误（如 `NoCredentials`）。
    *   **自动路径解析**：支持 `smb://` 前缀自动剥离及端口号智能识别。
*   **多服务器管理**：支持保存多个服务器配置，一键切换不同漫画库。

## 🖼️ 2. 智能浏览与导航 (Smart Browsing)

*   **瀑布流缩略图 (Staggered Grid)**：主界面以瀑布流形式展示图片，美观且空间利用率高。
*   **双指缩放网格 (Pinch-to-Zoom Grid)**：在主界面支持通过双指捏合动作动态调整网格列数（如从 2 列变更为 5 列），实时改变预览尺寸。
*   **深度文件夹导航**：
    *   支持无限层级的子文件夹递归浏览。
    *   **自动路径追踪**：清晰记录当前所处目录层级，方便快速返回。
*   **连接测试功能**：在配置服务器时，可实时调用 Rust 核心进行连接测试并即时列出根目录文件，确保配置正确。

## 📖 3. 专业阅读模式 (Professional Reading Modes)

*   **Webtoon 模式 (垂直滚动)**：
    *   专为长条漫设计，支持垂直方向连续滚动。
    *   **宽度优先策略**：针对 WebDAV 流式加载优化，优先保证图片宽度铺满屏幕且清晰，拒绝模糊。
*   **卡片模式 (Card Mode / 翻页)**：
    *   传统的左右滑动翻页，适合单张插画或日漫排版。
*   **沉浸式阅读 (Immersive Mode)**：
    *   自动隐藏系统状态栏与底部导航栏。
    *   支持点击屏幕唤出/隐藏 UI 控件。

## ⚡ 4. 性能与底层架构 (Performance & Architecture)

*   **Rust 高性能核心 (`rust-core`)**：
    *   使用 `tokio` 异步运行时处理所有网络 I/O，不阻塞 UI 线程。
    *   使用 `opendal` 和 `smb` 库提供工业级的协议支持。
*   **深度定制 Glide 加载引擎**：
    *   **流式加载 (Streaming)**：直接从 WebDAV/SMB 获取数据流传递给 Glide，无需先下载完整文件。
    *   **智能缓存策略**：支持针对远程图片的磁盘缓存，二次打开秒加载。
*   **UniFFI 跨语言调用**：通过 UniFFI 实现 Kotlin 与 Rust 的高效交互，保证核心逻辑的一致性。

## 🛠️ 5. 系统管理与工具 (System Tools)

*   **日志管理系统 (LogManager)**：内置调试级别控制，方便用户在遇到连接问题时导出日志进行排查。
*   **主题定制**：支持 Material Design 3 动态配色，提供主题预览与切换功能。
*   **文件夹缓存 (Room Database)**：使用 Room 数据库存储已访问的文件夹结构，加速二次访问速度。
