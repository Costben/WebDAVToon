# WebDAVToon 用户手册

WebDAVToon 是一款高性能的漫画与图片浏览器，支持本地存储和远程协议（WebDAV/SMB）。得益于 Rust 核心库的加持，它提供了极速的连接体验和强大的兼容性。

## 📥 安装与启动

1. 下载最新版本的 `app-debug.apk`。
2. 安装到 Android 设备（需开启“允许安装未知来源应用”）。
3. 启动应用，授予**存储权限**（用于读取本地图片和缓存远程文件）。

## 🚀 连接远程服务器

WebDAVToon 支持两种主流的远程文件协议：**WebDAV** 和 **SMB**。

### 1. 进入设置
点击主界面右上角的 **齿轮图标 (⚙️)** 进入设置页面。

### 2. 配置 WebDAV
适用于 AList、群晖 NAS、Nextcloud 等。
- **服务器地址**: 输入完整的 URL，例如 `http://192.168.1.100:5244/dav`。
- **用户名**: 您的 WebDAV 用户名。
- **密码**: 您的 WebDAV 密码。
- **点击“测试连接”**:
  - 成功：会弹出一个对话框，列出服务器根目录下的文件列表。这是连接成功的确切证明。
  - 失败：会显示错误信息（如 401 Unauthorized 或 Connect Timeout）。

### 3. 配置 SMB (Samba)
适用于 Windows 共享文件夹、NAS SMB 服务等。
- **主机**: 输入 IP 地址或主机名，例如 `192.168.1.100` 或 `MY-NAS`。
  - *注意：不要加 `smb://` 前缀，也不要加路径。*
- **共享文件夹**: 输入共享名称，例如 `Public` 或 `Comics`。
- **用户名**: SMB 用户名。
- **密码**: SMB 密码。
- **点击“测试连接”**:
  - 成功：弹出根目录文件列表对话框。
  - **重要说明**:
    - 本应用使用 **SMB v3** 协议 (3.0 - 3.1.1)，安全性更高。
    - 认证方式强制使用 **NTLM**，解决了 Android 平台上常见的 "NoCredentials" 错误。
    - 确保您的服务端开启了 SMB v3 支持。

## 📖 浏览模式

### 瀑布流主页 (Gallery)
- **手势**: 双指捏合/张开可调整列数（2列 - 5列）。
- **排序**: 在设置中可更改排序方式（名称、日期）。

### 阅读器 (Reader)
点击任意图片进入阅读模式。
- **普通模式 (Card)**: 左右滑动翻页，适合单幅插画。
- **条漫模式 (Webtoon)**: 垂直连续滚动，无缝拼接，专为韩漫/条漫优化。
  - 点击屏幕中央可唤出/隐藏菜单。
  - 沉浸式阅读：自动隐藏状态栏。

## ❓ 常见问题排查

**Q: SMB 连接提示 "Sspi error: NoCredentials"?**
A: 请确保您使用的是最新版本的 App。我们已在底层强制启用了 NTLM 认证并禁用了 Kerberos，完美修复了此问题。

**Q: WebDAV 连接失败?**
A:
1. 检查 URL 是否包含 `/dav` 等后缀（视服务端而定）。
2. 确认手机与服务器在同一局域网（或公网 IP 可达）。
3. 使用“测试连接”功能查看具体报错。

**Q: 图片加载慢?**
A: 应用内置了 Rust 多线程并发加载和 Glide 缓存。初次加载可能需要从服务器下载，再次浏览会非常快。

**Q: 核心库初始化失败?**
A: 如果遇到 "Rust Core Init Failed"，通常是架构不匹配。本应用目前主要支持 `arm64-v8a` 和 `x86_64` 架构。请确保下载了对应的 APK。
